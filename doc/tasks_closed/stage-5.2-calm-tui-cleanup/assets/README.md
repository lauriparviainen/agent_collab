# Assets

This folder stores visual references for the calm TUI cleanup task.

`grok-cli-reference.svg` is a lightweight vector reference recreated from the
user-provided Grok CLI screenshot in chat. The exact uploaded PNG was not
available as a local file in this session, so this SVG preserves the observed
layout structure for planning:

- top context/status line,
- user message band,
- subdued assistant/thinking metadata,
- bottom slash-command palette,
- bottom input rail with mode/status hints.

If the original screenshot becomes available as a file, place it here as
`grok-cli-reference.png` and keep the SVG as an annotated planning copy.
